# Model Habitat — 鲌类栖息地适宜性建模

> **触发**: 用户查询鲌类栖息地、产卵场、水位影响、HSI
> **方法**: HSI 模型 + 环境因子分析 + 产卵场评估

## PREFLIGHT

1. 确认目标物种及其生活史阶段 (成鱼/产卵/仔鱼/越冬)
2. 收集环境数据: 水深、水温、透明度、流速、溶解氧、水生植被
3. 确定建模空间范围和时间分辨率

## HSI 模型框架

```
HSI = (SI₁ × SI₂ × ... × SIn)^(1/n)           # 几何平均法
HSI = min(SI₁, SI₂, ..., SIn)                  # 最小值法 (限制因子)
HSI = w₁·SI₁ + w₂·SI₂ + ... + wn·Sin           # 加权平均法
```

## 关键环境因子 (翘嘴鲌)

```
1. 水深 (Depth):
   - 最适: 2-6 m
   - SI: 钟形曲线

2. 水温 (Temperature):
   - 最适: 22-28°C
   - 产卵阈值: > 20°C
   - SI: 偏正态分布

3. 透明度 (Secchi depth):
   - 最适: 40-80 cm (湖泊)
   - 翘嘴鲌为视觉捕食者，对透明度敏感

4. 溶解氧 (DO):
   - 阈值: > 4 mg/L
   - SI: S型曲线

5. 水生植被覆盖度:
   - 产卵场: 20-50%
   - 索饵场: < 30% (开阔水域)
```

## 产卵场评估

```
产卵场需求:
  - 底质: 水草/挺水植物附着基质
  - 流速: < 0.3 m/s (静水或缓流)
  - 水深: 0.5-3 m
  - 水温: 20-28°C (持续 > 5天)

水位波动影响:
  - 水位下降 → 产卵场裸露 → 繁殖失败
  - 水位骤涨 → 卵被冲刷 → 孵化率降低
  - 建议: 4-7月生态调度，维持水位稳定
```

## 栖息地破碎化评估

```
阻隔因子:
  - 闸坝: 阻断产卵洄游
  - 围垦: 减少沿岸带栖息地
  - 航道: 干扰 + 栖息地切割

连通性指标:
  - 纵向连通性: 闸坝密度 (座/100km)
  - 横向连通性: 江湖阻隔率 (%)
```
