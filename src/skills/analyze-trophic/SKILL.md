# Analyze Trophic — 鲌类同位素与营养生态位

> **触发**: 用户查询鲌类同位素、δ13C、δ15N、营养级、食性、胃含物
> **方法**: 稳定同位素分析 + MixSIAR 混合模型 + SIBER 生态位宽度 + 胃含物分析

## PREFLIGHT

1. 确认目标物种、水体、采样年份
2. 确认可用数据: 同位素值 / 胃含物 / 基线生物
3. 选择合适的混合模型和基线

## 稳定同位素分析流程

```
STEP 1: 样品处理
  - 组织: 背部白肌 (慢转换, 反映数月整合食性)
  - 脱脂: 氯仿-甲醇 (2:1) — 去除脂质对 δ¹³C 的影响
  - 冻干 + 研磨至均匀粉末
  - 称量: ~0.5-1.0 mg 入锡杯

STEP 2: EA-IRMS 测定
  - δ¹³C: 精度 ±0.1‰
  - δ¹⁵N: 精度 ±0.2‰
  - 标准: VPDB (δ¹³C), AIR (δ¹⁵N)

STEP 3: 质量保证
  - 每 10 样品插入 1 个标准品
  - C/N 比: 3.0-3.5 (肌肉; 若 > 4.0 需重新脱脂)
```

## 营养级估算

```
公式: TL = λ + (δ¹⁵N_consumer - δ¹⁵N_baseline) / Δδ¹⁵N

参数:
  λ   = 基线生物营养级 (初级消费者 = 2)
  Δδ¹⁵N = 每营养级富集因子 (2.5-3.4‰, 常用 3.4‰)

基线选择:
  沿岸带 (littoral): 螺类 (Bellamya aeruginosa), 刮食者
  湖心带 (pelagic): 蚌类 (Corbicula fluminea), 滤食者
  混合基线: 双基线 SIBER 模型
```

## 混合模型 — MixSIAR

```
R 包: MixSIAR (Bayesian mixing model)

模型输入:
  - 消费者 δ¹³C, δ¹⁵N (均值 ± SD)
  - 饵料源 δ¹³C, δ¹⁵N (均值 ± SD) — 需 TEF 校正
  - TEF (trophic enrichment factor):
      δ¹³C: 0.4 ± 1.3‰ (均值 ± SD)
      δ¹⁵N: 3.4 ± 1.0‰

模型输出:
  - 各饵料源后验贡献比例 (中位数 + 95% 可信区间)
  - 模型诊断: Gelman-Rubin, Geweke

示例饵料源:
  - 鱼类 (aggregate): δ¹³C -22‰, δ¹⁵N 14‰
  - 虾类: δ¹³C -23‰, δ¹⁵N 11‰
  - 水生昆虫: δ¹³C -25‰, δ¹⁵N 8‰
```

## 生态位宽度 — SIBER

```
R 包: SIBER (Stable Isotope Bayesian Ellipses in R)

指标:
  - SEA (标准椭圆面积, ‰²): 核心生态位宽度
  - SEAc (小样本校正 SEA)
  - 椭圆重叠率 (%): 种间生态位重叠

比较分析:
  - 不同水体: 湖泊 > 水库 > 河流?
  - 不同年: 丰水年 > 枯水年?
  - 不同种: C. alburnus vs C. mongolicus
  - 不同体长级: 幼鱼 vs 成鱼
```

## 胃含物分析 (互补)

```
指标:
  - 出现频率 (F%): 含某饵料的胃数 / 总胃数 × 100
  - 个数百分比 (N%): 某饵料个体数 / 总饵料数 × 100
  - 重量百分比 (W%): 某饵料重量 / 总饵料重 × 100
  - IRI (Index of Relative Importance): (N% + W%) × F%
  - 空胃率: 空胃数 / 总胃数

食性重叠指数:
  - Pianka (1973): O_jk = Σ(p_ij × p_ik) / √(Σp_ij² × Σp_ik²)
  - Schoener: α = 1 - 0.5 × Σ|p_ij - p_ik|
  取值范围: 0 (完全分离) ~ 1 (完全重叠)
  ≥ 0.6: 显著重叠
```

## 个体发育食性转换

```
阶段划分 (基于体长):
  仔鱼期 (< 2 cm):   轮虫 + 枝角类
  稚鱼期 (2-10 cm):  桡足类 + 水生昆虫幼虫
  幼鱼期 (10-30 cm): 虾类 + 小型鱼类
  成鱼期 (> 30 cm):  纯鱼类为主

同位素证据:
  δ¹³C: 随体长增加而升高 (沿岸带饵料贡献增加)
  δ¹⁵N: 体长 > 30 cm 后跳跃式升高 ~2‰
```
